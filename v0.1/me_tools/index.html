MatrixExponential(

    S::Array{Float64,2},
    s::Array{Float64,1},
    D::Array{Float64}=[0],
)

Inputs:

  • a a 1 by p Array of reals
  • S a p by p Array of reals
  • s a p by 1 Array of reals
  • D an optional argument, if empty then me.D is the identity, else is a p by p matrix

Throws an error if the dimensions are inconsistent.
