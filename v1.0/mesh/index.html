Mesh (discretisation)-related Types

DiscretisedFluidQueues.Mesh — Type
Mesh

Abstract type representing a discretisation mesh for a numerical scheme. Possible subtypes are DGMesh, FRAPMesh, FVMesh

source
DiscretisedFluidQueues.DGMesh — Type
DGMesh{T} <: Mesh{T}

A structure representing a discretisation scheme to be used for a DiscretisedFluidQueue.

Arguments:

  • nodes::AbstractArray{Float64, 1}: The edges of the cells.
  • n_bases::Int: The number of basis functions used to represent the solution on each cell
source
DiscretisedFluidQueues.FRAPMesh — Type
FRAPMesh{T} <: Mesh{T}

A QBD-RAP discretisation scheme for a DiscretisedFluidQueue.

Arguments

  • nodes: the cell edges
  • me: the MatrixExponential used to approximate model the fluid queue on each cell.
source
DiscretisedFluidQueues.FVMesh — Type
FVMesh{T} <: Mesh{T}

A finite volume discretisation scheme for a DiscretisedFluidQueue.

Arguments

  • nodes: the cell edges (the nodes are at the center of the cell)
  • order: the order of the polynomial interpolation used to approximate the flux at the

cell edges.
